# Quickstart Install

Download it off github

```
git clone https://github.com/devincapriola/pegheads.org.git
```

&nbsp;

Cd into the file

```
cd ./pegheads.org
```

&nbsp;

Install npm

```
npm install
```

&nbsp;

Run the app

```
npm start
```
